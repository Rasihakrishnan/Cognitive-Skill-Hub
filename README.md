# NeuroHire – Cognitive Skill Assessment Platform

**Full-Stack Project (AI-assisted Prototype, Rebuilt for Learning)**  
**Frontend:** React + Tailwind CSS | **Backend:** Spring Boot + MySQL

---

## **Project Overview**

NeuroHire is a cognitive skill assessment platform designed to evaluate logic and memory-based skills.  

- Users can log in, take quizzes, and see their scores.  
- Admins can create quizzes, add questions, and view user results.  
- Leaderboard functionality ranks top performers.  

> ⚡ Initially prototyped using AI on Replit, then **fully rebuilt locally** to master full-stack development.

---

## **Features**

- **Login System:** Email & password authentication  
- **Admin Dashboard:** Create quizzes, add questions, view results  
- **Dynamic Quiz Page:** Multiple-choice questions with progress tracking  
- **Submit & Score:** Automatic scoring system  
- **Leaderboard:** Displays top scores  
- **AI Hints (Optional):** Sample placeholders for cognitive questions  

---
